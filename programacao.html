Minicurso 02: Versionamento de arquivos e projetos com GIT

Palestrante: Rafael Campo Nunes

Descrição: O minicurso tem objetivo de explicar o que é um versionador de arquivos e suas aplicações. Além disso, introduz conceitos fundamentais para utilização de qualquer versionador de arquivo, apresentando também o Git como opção e como utilizá-lo. Serão trabalhados conceitos básicos de versionamento como o gerenciamento de commits, múltiplos branches de desenvolvimento e um possível fluxo de trabalho no ambiente corporativo.

Horário: 13:30 - 17:30

Data: 25 de setembro de 2017
